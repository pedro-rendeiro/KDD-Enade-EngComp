# KDD-HigherEduBR

Data Mining in Brazilian Higher Education

### Prerequisities

* Python 3.6.9+
* Linux

Install required packages by:

```bash
pip install -r requirements.txt
```

### Usage

#### Get the data

```bash
cd data/
bash get_data.sh
```

#### View/Run the analysis

```
jupyter notebook main.ipynb
```

#### Running the Tests

```bash
pytest-3 tests/
```
